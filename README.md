# 🏠 Flatmate – Find Your Perfect Room Partner

Flatmate is a platform designed to help people easily find compatible flatmates and shared accommodations without the hassle of random searching.

---

## 🚀 Problem Statement

Finding a flatmate is often:
- Unorganized (WhatsApp, Facebook groups)
- Unsafe (no verification)
- Time-consuming
- Lacking proper communication and tracking

Flatmate solves this by providing a **structured, secure, and user-friendly platform**.

---

## 💡 Solution

Flatmate allows users to:
- Post available flats/rooms
- Search for flats based on location
- Connect with potential flatmates via in-app chat
- Lock deals securely without sharing personal numbers

---

## ✨ Features

### 👤 User Authentication
- Secure login/signup
- Profile creation with basic details

### 🏢 Flat Listing
- Add flat details:
  - Location
  - Rent
  - Room type (1BHK, 2BHK, etc.)
  - Images
  - Description

### 🔍 Smart Search
- Search flats by:
  - Area
  - Budget
  - Preferences

### 💬 In-App Chat
- Communicate without sharing phone numbers
- Safe interaction between users

### 🔒 Deal Lock System (Unique Feature)
- Both users confirm the deal using IDs
- Status changes to **"Booked"**
- Prevents misuse and duplicate listings

### 📊 User Dashboard
- View:
  - Your listings
  - Interested users
  - Chat history

---

## 🧠 How It Works

1. User signs up and creates a profile  
2. Adds flat details and posts listing  
3. Other users browse/search listings  
4. Interested users connect via chat  
5. Both parties confirm → deal gets locked  
6. Listing marked as **"Booked"**
